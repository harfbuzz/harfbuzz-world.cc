Welcome

HarfBuzz World is a live playground for HarfBuzz, a small set of in-browser demos backed by the same C/C++ library that ships in browsers, OSes, and design tools. The shared Text, Size, and Font controls above feed every demo:

  • shape — the shaped glyph stream as JSON, alongside an SVG preview.
  • subset — produce a font subset for the current text and download it.
  • raster — pixel-perfect BGRA rendering at any size, blitted to a canvas.
  • vector — the same shaped text rendered to SVG and downloadable PDF.
  • gpu — slug-based GPU rendering via WebGL2 / WebGPU.

Embed in your own site

The wasm bundle is two files — hb-world.js and hb-world.wasm — and exposes a small C surface for shaping and rendering. Drop them next to your page:

<script src="hb-world.js"></script>
<script>
  const Module = await createHbWorld ();
  const fontPtr = Module._malloc (fontBytes.length);
  Module.HEAPU8.set (fontBytes, fontPtr);
  const textPtr = Module._malloc (Module.lengthBytesUTF8 (text) + 1);
  Module.stringToUTF8 (text, textPtr, /* maxBytes */ 1024);
  const svgPtr = Module._web_render_svg (fontPtr, fontBytes.length,
                                         textPtr, /* px */ 48);
  document.body.innerHTML = Module.UTF8ToString (svgPtr);
  Module._web_free_string (svgPtr);
</script>

Source and build scripts: github.com/harfbuzz/harfbuzz-world.cc.
